item 전용 속성

order : item의 순서를 지정하는 속성

    -item이 중심축(main axis)에 배치되는 순서를 지정
    -모든 item의 order기본값은 0
    -작성 할 수 있는 값은 정수 형태(+, 0, -)
    -값이 클 수록 중심값 끝 쪽에 배치됨
    -동일한 order 값을 갖는경우 html배치순서를 따른다.

flex-grow : item 팽창 

    item이 flex-container 내부에서 중침축 방향 
    비어있는 공간을 팽창하여 메꾸는 정도를 지정하는 속성

    -item의 flex-grow기본값 : 0 (팽창X)

    -item중 flex-grow 값이 1 이상인 요소가 1개인 경우
    ->해당 item이 빈 공간을 모두 메꿈(혼자만 팽창)

    -item 중 flex-grow 값이 1 이상인 요소가 n개인 경우
    -> 지정된 값 만큼의 비율로 나눠서 팽창

flex-shirink : item 수축

    - item이 수축되는 정도를 지정

    - flex-shirink 기본 값 : 1

    - 0 으로 설정-> item 수축 X

    - n개의item에 flex-shirink값이 1이상으로 설정되어있는 경우
    -> 지정된 값 만큼의 비율로 나눠서 수축

flex-basis : 중심축 방향 크기지정

    flex-container의 설정된 중심축 방향으로
    item의 기본 점유율(크기)를 지정하는 속성

    -각종 크기 단위 사용 가능
    (px, %, vh, em,rem 등)
